Level 4

Features

  • The level consists of a multi-player tank simulation. (done)
  • The level includes a 3D environment with walls in places. (done)
  • Each player is represented by a tank. (done)
  • Collisions should be detected between the tank and the walls. (done)
  • Each tank can be made of basic shapes (e.g., cylinders, and cubes) (done)
  • Each tank can rotate and move forward and back. (done)
  • Each tank can fire missiles. (done)
  • Each player will be instantiated using a Round-Robin method. (done)
  • Each tank is destroyed after being hit 10 times.
  • Once a tank has been destroyed it will be re-instantiated at a location determined by a Round-Robin method.
  • Each player should include a camera that follows its movement (i.e., top-view). (done)
  • Each player should avail of a mini-map. (done)
  • Once the player has collected all the relevant objects, the game ends and the player has won.

Marking

  • <40% (fail): the level can't be played by 2+ players simultaneously.
  • 40-50%: The level can be played by 2+ players simultaneously and each player is represented by a 3D object that can be moved accordingly.
  • 51-70%: Each tank is instantiated using a Round-Robin method and can fire missiles and collide with walls.
  • 71-80%: Each player avails of a mini-map, a camera that follows its movement using a top-down view.
  • 80%+: Missiles produce an explosion upon collision.
